Router

Request & Response
Simple wrappers for reading input and sending consistent JSON responses.

Quick start
Up and running in seconds
Clone, install, and start building your API right away.
# Clone & install
git clone https://github.com/MYusron12/artemis.git
composer install

# Start the server
php artemis run

# Define your routes in routes/api.php
$router->get('/users', [UserController::class, 'index']);
$router->post('/users', [UserController::class, 'store']);
$router->get('/users/{id}', [UserController::class, 'show']);

# Response format
{
  "responseCode": "200M500",
  "responseMessage": "Successful",
  "data": []
}
