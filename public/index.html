Core features
Everything you need, nothing you don't
Artemis ships only the essentials — small enough to understand fully, powerful enough for real projects.
Router
Clean URL routing with dynamic parameters, HTTP methods, and version grouping.
Request & Response
Simple wrappers for reading input and sending consistent JSON responses.
Middleware
Run logic before or after your controllers — auth, logging, and more.
Database
Lightweight query builder — no complex ORM required.
CLI Tool
Run your server, generate files, and migrate with simple commands.
Quick start
Up and running in seconds
Clone, install, and start building your API right away.
# Clone & install
git clone https://github.com/MYusron12/artemis.git
composer install

# Start the server
php artemis run
# Running at http://localhost:8300

# Define your routes in routes/api.php
$router->group('/openapi/v1.0', function($router) {
    $router->get('/users', [UserController::class, 'index']);
    $router->post('/users', [UserController::class, 'store']);
    $router->get('/users/{id}', [UserController::class, 'show']);
    $router->put('/users/{id}', [UserController::class, 'update']);
    $router->delete('/users/{id}', [UserController::class, 'destroy']);
});
