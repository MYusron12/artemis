Core features
Everything you need, nothing you don't
Artemis ships only the essentials — small enough to understand fully, powerful enough for real projects.
Router
Clean URL routing with dynamic parameters, HTTP methods, and version grouping.
Request & Response
Simple wrappers for reading input and sending consistent JSON responses.
Middleware
Auth, logging, rate limiting, CORS — all as composable middleware.
Database
Lightweight query builder with SQLite and MySQL support.
Validation
Simple input validation — required, email, min, max, numeric.
Logging
Daily log files — requests, errors, duration, and custom messages.
CLI Tool
Run, migrate, test, and generate files with simple commands.
SNAP BI
Asymmetric & Symmetric Signature, Access Token B2B — ready to use.
Quick start
Up and running in seconds
Clone, install, and start building your API right away.
# Clone & install
git clone https://github.com/MYusron12/artemis.git
composer install

# Generate RSA keys (for SNAP BI)
php generate_keys.php

# Setup environment & migrate
cp .env.example .env
php artemis migrate

# Start the server
php artemis run
# Running at http://localhost:8300

Available endpoints

Users
GET/openapi/v1.0/users Retrieve all users▼
cURL
Response
curl -X GET http://localhost:8300/openapi/v1.0/users \
  -H "Content-Type: application/json"
Response 200
{ "responseCode": "200M500", "responseMessage": "Successful", "data": [{ "id": 1, "name": "Budi", "email": "budi@mail.com", "created_at": "2026-04-12 10:00:00" }] }
POST/openapi/v1.0/users Create a new user▼
cURL
Response
curl -X POST http://localhost:8300/openapi/v1.0/users \
  -H "Content-Type: application/json" \
  -d '{"name": "Citra", "email": "citra@mail.com"}'
Response 201
{ "responseCode": "201M500", "responseMessage": "Successful", "data": { "id": 1, "name": "Citra", "email": "citra@mail.com" } }
Response 400
{ "responseCode": "400M502", "responseMessage": "Invalid Mandatory Field name" }
Response 409
{ "responseCode": "409M509", "responseMessage": "Data already exists" }

SNAP BI
GET/snap/v1.0/get-token Generate asymmetric signature▼
cURL
Response

Response 200
{ "responseCode": "200M500", "responseMessage": "Successful", "data": { "clientId": "artemis-dummy-client-001", "timestamp": "2026-04-12T07:53:29+07:00", "signature": "abc123..." } }
POST/snap/v1.0/access-token/b2b Request Access Token B2B▼
cURL
Response
curl -X POST http://localhost:8300/snap/v1.0/access-token/b2b \
  -H "Content-Type: application/json" \
  -H "X-CLIENT-KEY: artemis-dummy-client-001" \
  -H "X-TIMESTAMP: {timestamp-from-get-token}" \
  -H "X-SIGNATURE: {signature-from-get-token}" \
  -d '{"grantType": "client_credentials"}'
Response 200
{ "responseCode": "200M500", "responseMessage": "Successful", "data": { "accessToken": "eyJ...", "tokenType": "BearerToken", "expiresIn": 900 } }
Response 401
{ "responseCode": "401M401", "responseMessage": "Unauthorized. Invalid Signature" }
